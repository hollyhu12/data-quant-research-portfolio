# Data Analytics and Quantitative Research Portfolio

This portfolio collects six research projects across quantitative finance, time series forecasting, econometrics, and mathematical optimization.

## Project Index

| Project | Area | Methods | Highlights |
| --- | --- | --- | --- |
| [SMA Stock Selection Strategy](projects/01_sma-stock-selection/README.md) | Quantitative finance | Fundamental factor scoring, technical indicators, SMA backtesting | Selected stocks achieved a mean annualized return of 107.17% versus 2.53% for random selection in the study period. |
| [Gold Price ARIMA Forecasting](projects/02_gold-price-arima/README.md) | Time series | Stationarity tests, ARIMA modeling, diagnostics, forecasting | Built a weekly gold price forecasting workflow with model selection and residual diagnostics. |
| [Education Expenditure Econometrics](projects/03_education-expenditure-econometrics/README.md) | Econometrics | Multiple regression, log transformation, VIF, BP/White tests, Durbin-Watson | Identified key regional factors associated with education expenditure and validated model reliability. |
| [A-Share Communications Equipment Risk and Sentiment](projects/04_a-share-communication-risk-sentiment/README.md) | Financial risk analysis | PCA risk scoring, Granger causality, ARIMA | Connected stock-level risk score variance with market sentiment and price volatility. |
| [Multibeam Sonar Survey Line Optimization](projects/05_multibeam-sonar-survey-optimization/README.md) | Mathematical modeling | 2D/3D geometry, local search, interpolation, optimization | Designed survey line layouts under coverage, overlap, and path-length constraints. |
| [Machine Learning Quant Stock Selection](projects/06_ml-quant-stock-selection/README.md) | Machine learning in finance | PCA, SVM, logistic regression, KNN, random forest, backtesting | ML strategies outperformed the CSI 500 benchmark during the backtest window. |
| [Long-Term Time Series Forecasting Benchmark](projects/07_long-term-time-series-forecasting-benchmark/README.md) | Deep learning and forecasting | DLinear, LightGBM, LSTM, PatchTST, benchmark evaluation | Compared four forecasting paradigms across ETTm1, Electricity, and Weather datasets. |

## Core Skills Demonstrated

- Quantitative finance strategy design and backtesting
- Time series modeling and forecasting
- Econometric modeling and diagnostic testing
- Machine learning model comparison and feature reduction
- Mathematical modeling and optimization
- Python, R, MATLAB, SPSS/Stata-style econometric analysis, Jupyter Notebook
- Deep learning and machine learning model benchmarking for time series forecasting

## Public Portfolio Notes

- Notebook credentials have been replaced with environment variable placeholders.
- Proprietary or licensed data exports, such as Wind database files, are not included in the public version.
- Original reports are included as PDFs inside each project folder.
