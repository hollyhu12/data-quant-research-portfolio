# Gold Price ARIMA Forecasting

## Overview

This project analyzes weekly gold price trends and builds an ARIMA-based forecasting model for future price movement.

## Research Question

Can ARIMA time series modeling capture weekly gold price dynamics and provide useful short-term forecasts?

## Methods

- Collected weekly gold price data from 2021 to 2024.
- Converted raw data into time series format.
- Conducted exploratory visualization and stationarity analysis.
- Applied ARIMA model selection and diagnostics.
- Forecasted future values and discussed model limitations.

## Files

- `gold_price_arima.ipynb`: Jupyter Notebook containing the time series workflow.
- `GoldPrice(week).csv`: Weekly gold price input data.
- `GoldPrice(week_all).csv`: Extended weekly gold price data.

## Resume Bullet

Built a weekly gold price forecasting workflow using ARIMA modeling, stationarity testing, residual diagnostics, and forecast visualization to support investment-oriented time series analysis.
